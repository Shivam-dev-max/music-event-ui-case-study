# Music Event Website UI/UX Case Study

## Overview
This is a modern music event website design for exploring concerts and booking tickets.

## Problem
Users find it difficult to discover and explore upcoming music events easily.

## Solution
Designed a clean and engaging interface with event cards, artist highlights, and simple navigation.

## Design Process
- Research
- Wireframing
- UI Design in Figma

## Features
- Event listing
- Artist showcase
- Ticket booking CTA
